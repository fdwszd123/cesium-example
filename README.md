# Cesium常见功能集合

项目使用vue3+vite+cesium搭建记录了学习cesium过程中实现的的一些常见功能。包括粒子特效、材质、量测工具、绘制工具等。具体功能如下所示。

## 项目启动

```
pnpm install
pnpm dev
```

## 功能（已完成）

  - [X] 天空盒更换
  - [X] 地形数据加载

  - [X] 绘制工具(点、线、面、矩形、圆)
